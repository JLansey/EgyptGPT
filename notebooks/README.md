# Notebooks

This folder holds local exploratory notebooks that were previously sitting in the repo root.

- `running_EgyptGPT.ipynb`: the main experimental notebook for running generation and translation steps.
- `review_translations.ipynb`: notebook for reviewing and rating translated outputs.

These notebooks are part of the local EgyptGPT workflow, but the generated CSV, JSONL, and text artifacts belong under `out/` by pipeline stage.
